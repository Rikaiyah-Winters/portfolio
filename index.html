My portfolio

U.S. Election Map

HTML5-CSS3-JavaScript

This is an interactive map of the results of the U.S. Election. The voting model is based on the U.S. Electoral College system where the winner is chosen based on the results from each state and total votes. Mousing over each state indicates which of the two candidates won each state. Creating this project provided great practice with refrencing code from pre-existing code files.

Magic 8-Ball

HTML5-CSS3-JavaScript-jQuery

"Why did the Magic 8-ball call tech support?
Outlook not good."
This Magic 8-Ball uses JavaScript and jQuery to provide simple answers to questions imput by users. jQuery functions such as prompt(), effect("shake"), and fadeIn() help create a lively interactive website that users will want to play with over and over again.

Lol Cat Clock

HTML5-CSS3-JavaScript

Lol Cat Clock uses JavaScript to: display a live clock, change the main image to different pictures of cats at different times of the day, and allow users to change times for "wake-up, lunch, and nap-time" so that the main image can change. This project provided a greater understanding of simple JavaScript and the DOM.

Unplugged

HTML5-CSS3-Flexbox-Photoshop

Unplugged is a website for a retreat center that uses an HTML5 boiler plate to make it viewable on a multitude of devices. Building this website also introduced the concepts of iframes and ifonts.
